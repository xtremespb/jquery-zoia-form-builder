zoiaFormBuilder Demo

zoiaFormBuilder let's you to build forms and takes responsibilty for data validation and event handlers.

This demo uses static data (e.g. there is no server processing for loading data to form), therefore it doesn't cover all features provided. You may take a look at the Zoia project which is actively using zoiaFormBuilder. Note: the Captcha widget is not available in demo!

Basic usage

Below is the basic usage example. The mandatory fields are "Username" and "Password", but if you type a wrong e-mail address, you will get an error, too.

  • Preview
  • Markup

  • HTML code:

    
<form id="form1"></form>

    Javascript code:

    
$('#form1').zoiaFormBuilder({
    save: {
        url: '#',
        method: 'GET'
    },
    template: {
        fields: '{fields}',
        buttons: '{buttons}'
    },
    events: {
        onInit: () => {},
        onSaveSubmit: () => {},
        onSaveValidate: () => {},
        onSaveSuccess: () => {},
        onSaveError: (res) => {}
    },
    items: {
        username: {
            type: 'text',
            label: 'Username',
            css: 'uk-form-width-medium',
            autofocus: true,
            helpText: 'Latin characters and numbers, length: 3-20',
            validation: {
                mandatoryCreate: true,
                length: {
                    min: 3,
                    max: 20
                },
                regexp: /^[A-Za-z0-9_\-]+$/,
                process: (item) => {
                    return item.trim().toLowerCase();
                }
            }
        },
        email: {
            type: 'email',
            label: 'E-mail',
            css: 'uk-width-medium',
            helpText: 'Example: user@domain.com',
            validation: {
                mandatoryCreate: false,
                length: {
                    min: 6,
                    max: 129
                },
                regexp: /^(([^<>()\[\]\\.,;:\s@"]+(\.[^<>()\[\]\\.,;:\s@"]+)*)|(".+"))@((\[[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}])|(([a-zA-Z\-0-9]+\.)+[a-zA-Z]{2,}))$/,
                process: (item) => {
                    return item.trim().toLowerCase();
                }
            }
        },
        password: {
            type: 'passwordConfirm',
            label: 'Password',
            helpText: 'Minimal length: 5 characters, type twice to verify',
            validation: {
                mandatoryCreate: true,
                length: {
                    min: 5,
                    max: 50
                },
                process: (item) => {
                    return item.trim();
                }
            }
        },
        buttons: {
            type: 'buttons',
            css: 'uk-margin-top',
            buttons: [{
                name: 'btnSave',
                label: 'Register',
                css: 'uk-button-primary',
                type: 'submit'
            }]
        }
    }
});

Widgets Showcase

Below is the form which contains all possible widgets. You may use one or more widgets in your form.

  • Preview
  • Markup

  • HTML code:

    
<form id="form2"></form>

    Javascript code:

    
$('#form2').zoiaFormBuilder({
    save: {
        url: '#',
        method: 'GET'
    },
    template: {
        fields: '{fields}',
        buttons: '{buttons}'
    },
    items: {
        username: {
            type: 'text',
            label: 'Text field',
            css: 'uk-form-width-medium',
            autofocus: true
        },
        email: {
            type: 'email',
            label: 'E-mail field',
            css: 'uk-width-medium'
        },
        password: {
            type: 'password',
            css: 'uk-width-medium',
            label: 'Password Field'
        },
        passwordConfirm: {
            type: 'passwordConfirm',
            label: 'PasswordConfirm Field'
        },
        select: {
            type: 'select',
            label: 'Select',
            css: 'uk-form-width-medium',
            values: {
                'value1': 'Value 1',
                'value2': 'Another value',
                'value3': 'Third (and last) value'
            },
        },
        textarea: {
            type: 'textarea',
            label: 'Textarea Field'
        },
        launcher: {
            type: 'launcher',
            label: 'Launcher Field',
            labelBtn: 'Select',
            value: 'Current Value'
        },        
        buttons: {
            type: 'buttons',
            css: 'uk-margin-top',
            buttons: [{
                name: 'btn1',
                label: 'Button 1',
                css: 'uk-button-primary',
                type: 'button'
            }, {
                name: 'btn2',
                label: 'Button 2',
                type: 'button'
            }]
        }
    }
});
